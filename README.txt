# AEONX Ventures Website

Modern static website for AEONX Ventures.

## Upload to GitHub Pages

Upload these files to the root of the `aeonxventures.github.io` repository:

- index.html
- styles.css
- script.js
- favicon.svg

The website will then appear at:

https://aeonxventures.github.io

## Important

The contact form is visual only right now. Connect it to Formspree, Web3Forms, or a business email service before launch.
